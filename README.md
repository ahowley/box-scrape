# Box SDK Project

This is a small node project for interfacing with the Box.com SDK & API.

## Requirements

-   Node.js

## Setup

-   Install Dependencies
    ```bash
    npm i
    ```
-   Set BOX_DEVELOPER_TOKEN=tokengoeshere in .env file, inside root directory (not in src/)
-   Run
    ```bash
    npm run dev
    ```
-   Run (watch)
    ```bash
    npm run watch
    ```
